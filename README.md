# Vue Facts App

A simple practice project built while learning **Vue.js**, showcasing some key facts about the framework. This project is part of my Vue learning journey and focuses on component structure, template syntax, and basic styling.

## Project Overview

The **Vue Facts App** presents short, informative facts about Vue.js, including its performance, ease of use, and versatility.

## Technologies Used

- Vue.js
- HTML5
- CSS3
- JavaScript (ES6+)

## Project Setup

```bash
# install dependencies
npm install

# run development server
npm run dev
```

## What I Practiced

- Vue component structure
- Template syntax & data binding
- Basic project organization
- Styling Vue applications
